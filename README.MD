## ğŸ§© Project Overview

RentalWheels is a responsive frontend web application designed to simplify the car rental experience. Users can explore available cars, view detailed pricing and provider information, and securely access protected features after authentication.

This repository contains the **frontend client** of the RentalWheels platform.

## âœ¨ Core Features

- Browse available rental cars
- View car details including pricing and ratings
- Secure user authentication using Firebase
- User profile with name and avatar
- Protected routes for logged-in users
- Light/Dark mode using DaisyUI themes
- Toast notifications for actions and errors
- Smooth animations and interactive UI
- Fully responsive across all devices
